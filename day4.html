With cupious example differentiate between comparisom and logical operators


  1. Comparisom Operator

    Comparison operators are used to compare two values or expressions. They return a Boolean value (true or false) based on whether the comparison is true or not.
    These operators help in determining relationships between values, such as equality, inequality, and relative magnitude.
    Here are the main comparison operators:

    • ==: Equal to
    • !=: Not equal to
    • ===: Strict equal to (compares both value and type)
    • !==: Strict not equal to (compares both value and type)
    • >: Greater than
    • <: Less than
    • >=: Greater than or equal to
    • <=: Less than or equal to
    Examples of Comparison Operators:


    let a = 10;
    let b = 20;
    let c = "10";

    // Example 1: Equal to (==)
    console.log(a == c); // true, because '10' is equal to "10" in value

    // Example 2: Strict equal to (===)
    console.log(a === c); // false, because '10' (number) is not equal to "10" (string) in type

    // Example 3: Not equal to (!=)
    console.log(a != b); // true, because 10 is not equal to 20

    // Example 4: Strict not equal to (!==)
    console.log(a !== c); // true, because 10 (number) is not strictly equal to "10" (string)

    // Example 5: Greater than (>)
    console.log(b > a); // true, because 20 is greater than 10

    // Example 6: Less than (<)
    console.log(a < b); // true, because 10 is less than 20

    // Example 7: Greater than or equal to (>=)
    console.log(a >= 10); // true, because 10 is equal to 10
    // Example 8: Less than or equal to (<=)
    console.log(a <= b); // true, because 10 is less than 20
  2. Logical Operators

    Logical operators are used to combine multiple Boolean expressions or to reverse the Boolean value.
    They are crucial in building complex conditional statements, where multiple conditions must be checked together.
    Here are the main logical operators:

    • &&: Logical AND (returns true if both operands are true)
    • ||: Logical OR (returns true if at least one operand is true)
    • !: Logical NOT (returns the opposite Boolean value)
    Examples of Logical Operators:


    let x = 5;
    let y = 10;
    let z = 15;

    // Example 1: Logical AND (&&)
    console.log(x < y && y < z); // true, because both conditions are true (5 < 10 and 10 < 15)

    // Example 2: Logical OR (||)
    console.log(x > y || y < z); // true, because the second condition (y < z) is true

    // Example 3: Logical NOT (!)
    console.log(!(x > y)); // true, because x > y is false, and !false is true

    // Example 4: Complex condition using AND and OR
    console.log((x < y && y > z) || (x == 5)); // true, because x == 5 is true

    // Example 5: Using NOT with AND
    console.log(!(x > 10 && y > 10)); // true, because x > 10 is false, and the entire AND condition is false, so !false is true
